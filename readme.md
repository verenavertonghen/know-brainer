## Know-brainer project
Ferre Lambert
Verena Vertonghen
3MCT - Web&UX
----------------------------------------------------

“Iedereen kan iets, niemand kan alles..
.. maar door kennis te delen, kunnen we onszelf en anderen iets leren.

Know-brainer is een sociaal platform waar mensen kennis kunnen delen of opdoen. 
Dit zou plaats vinden in Antwerpen. 
Kennis gaande van koken, naar bv. een unieke stadswandeling tot leren breien 
of heel specifiek: forms kunnen maken met een php-framework.
